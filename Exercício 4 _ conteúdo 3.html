Explicação para os elementos: head, body, p e section.


Elemento 1:

<head>: Permite definir vários tipos de infomação sobre o documento HTML. Entre essas tags, deve-se especificar, no mímimo, o elemento title e o elemento meta.

Elemento 2:

<body>: Permite estruturar o conteúdo a ser exibido na página HTML. Esse conteúdo pode ser descrito como texto, listas, tabelas, imagens, vídeos, formulários etc.

Elemento 3:

<p>: Permite criar parágrafos em uma página HTML.

Elemento 4:

<section>: Permite agrupar conteúdos relacionados. Pode haver várias seções em uma página HTML, e é uma boa prática definir um título para cada seção.
